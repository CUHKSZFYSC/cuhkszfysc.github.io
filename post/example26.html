Connection在申请PhD时的重要性 ​

我拿到纽约大学 PhD 的经历，可以说全程都是 connection 在发挥作用。

故事要从大二说起。那年暑假我去芝加哥做暑研，跟着导师的一位 postdoc 合作。那时候我们就认识了。后来这位 postdoc 去做了 assistant professor，我当时找不到暑研位置，看到他官网正在招人，就直接联系他——因为我们之前合作过，领域也相似，他就答应了。

说实话，他并不是我的首选，但他有一个巨大的优势：他在纽约。因为地缘之便，他能直接去 NYU 拜访别人，于是他把一位 NYU 的大佬邀请到我们的合作中来。那篇文章最终没中，但通过这次合作，我和那位大佬认识了。

申请 NYU 的时候，我给那位大佬发了很多封套磁邮件，他一封都没回。我是靠自己的 background 通过了 committee 的筛选和面试。然后他才突然联系我说："我觉得我现在应该来跟你聊聊。"

更神奇的是，最后给我发 offer 的并不是那位大佬，而是 NYU 另一位老师。他全程没有面试我就直接录取了我。后来我才知道——我的两位推荐人他都认识，加上我之前跟 NYU 的人合作过，他就直接给了 offer。

这一切的 connection 链条：芝加哥暑研 → 认识 postdoc → postdoc 当 AP 后做暑研 → 通过地缘认识 NYU 大佬 → 合作论文 → 申请时导师之间的认识 → 拿到 offer。每一环都是 connection。

不止是我：身边同学的故事 ​

回头看身边的人，几乎无一例外，大部分同学都通过本科期间的老师以及科研合作者拿到了offer。

我在admitted student的campus visit时认识到的同学也大多先前就和老师合作过或是有很强的推荐信。

面板只是门槛，Connection 才是关键 ​

我面试的时候，没有一个人跟我聊简历上的东西——没人聊成绩、没人聊上过什么课、没人聊 CV 和 personal statement。他们只让我介绍自己的 project 和想做什么。

GPA、课程这些东西，只要不拉就行。对统计项目来说比较看成绩，对 CS 方向则不那么在意。Publication 当然有帮助，但我自己也只有一篇一作会议论文，不是传统计算机三大会。这些面板数据是必要条件，但远不是充分条件。

申请策略：像 Graph 一样思考 ​

如果把经验转化成策略，我觉得应该这样想：

把学术界想象成一个图（Graph），每个人都是节点，connection 是边。你要最大化自己的机会，就应该往最有关联的方向延伸：

1.  同领域的老师——研究兴趣 match 是最基本的 connection

2.  合作过的老师/学校——暑研导师、合作论文的 co-author

3.  导师的导师——学术血脉往上追，他们的学校通常都很好

4.  导师的同门/postdoc——同一个学术网络里的节点

而不是在网上随机找导师发套磁信——那是 random 的，成功率极低。

学术会议：一个巨型 Connection 中枢 ​

我想特别讲讲学术会议的价值，因为很多本科生对此完全不了解。

我去年去了圣地亚哥的 NeurIPS（三大顶会之一），3 万多人参加。这不仅仅是一个学术会议——它是一个巨型的 connection 中枢：

● 本科生在那找 PhD

● 应届生在那找工作、现场面试

● 导师在那招学生、跟学生现场聊

● 学术人在那谈合作

● 业界的人逛 poster，找合作机会

世界上最好的公司几乎都在那设了摊位——Meta、Google、微软、Tesla、英伟达……你可以直接跟他们的员工交流。我见过 Tesla 的工程师被 Meta 和 OpenAI 的工程师围着问机器人技术细节...

参会费大概一两千，下一届 NeurIPS 在首尔，离中国很近，如果有机会真的值得去体验一下。

会议社交的经验 ​

在这种场合，被 ignore 是很正常的。我给想申请的导师发了很多封套磁邮件，基本都没人回。在现场想和导师 coffee chat，除了暑研导师的博导以外都没成功——包括后来给我发 offer 的 NYU 那位老师，我见到他的时候他在打电话，让我第二天再来，结果也没聊上。

但我认识的一个同学做得比我好很多：他打印了好几份简历和一个二维码，在现场直接递给感兴趣的老师。这种做法在学术会议上非常常见，也是有效的。

关键建议：准备好你的自我介绍。为不同的人准备不同版本：

1.  对同领域的 expert——从技术角度切入

2.  对同 department 但不同领域的人——从你解决的问题切入

3.  对 general audience——更通俗的版本

而且，你不一定非要介绍自己是本科生、什么专业——这些信息对抓住对方注意力没有帮助，甚至可能是负面的。重要的是在短短几分钟内，讲清楚你做了什么、你想做什么。

关于港中深的信息现状 ​

港中深在校友网络和信息传播方面确实偏弱。

很多在海外读博、参加顶会的港中深校友，其实表现非常出色。我在圣地亚哥的 poster session 旁边，过来问的人好几个都是港中深毕业生，在各种顶尖学校读博。但因为他们是先读硕再读博，学校里的人就不知道他们后来去了哪。

这也是飞跃手册存在的意义——把这些信息连接起来，让后面的人能看到真实的全貌。

给学弟学妹的话 ​

● 多跟人聊，尤其是跟导师聊——告诉他你想去哪，他可能会说"哦，我认识那边的谁"。很多 connection 是 intrinsic 的，只有聊了才能浮现。

● 主动争取机会。很多机会不会主动来找你，被 ignore 是常态，但只要有一次成功就值了。

● 学术会议值得去，哪怕你没有 paper，去了也能认识很多人、获得一手信息。

● 能力是必要条件，connection 是充分条件。先把自己的 background 做好（这是门槛），然后尽可能拓展和利用你的学术网络。
